# 📇 나만의 3D 디지털 명함 & 이력서 웹 사이트 (Netlify 배포 가이드)

이 프로젝트는 비전공자 및 코딩 초보자분들을 위해 특별히 맞춤 제작된 **프리미엄 3D 디지털 명함 및 이력서 웹사이트**입니다. PC와 모바일 화면 모두에서 부드러운 Glassmorphism 카드 디자인과 3D 뒤집기 인터랙션을 제공합니다.

---

## 📂 파일 구성
이 폴더는 인터넷에 즉시 배포할 수 있는 5개의 파일로 이루어져 있습니다:
*   `index.html`: 명함과 이력서의 뼈대와 글자 내용이 들어 있습니다.
*   `style.css`: 반투명 유리 효과 및 3D 회전 애니메이션 등 디자인이 들어 있습니다.
*   `script.js`: 버튼 클릭 시 회전 동작 및 마우스 무브 반응을 명령하는 코드가 들어 있습니다.
*   `profile.png`: 명함과 이력서에 나타나는 동그란 프로필 캐릭터 이미지입니다.
*   `README.md`: 본 배포 설명서 파일입니다.

---

## 🚀 넷리파이(Netlify Drop)를 이용한 5초 배포 가이드 (가장 추천)

로그인이나 프로그램 설치 없이 마우스 클릭과 드래그만으로 즉시 배포할 수 있습니다.

### 단계 1: 배포 사이트 접속
인터넷 창(크롬 등)을 열고 아래 링크로 이동합니다.
👉 **[https://app.netlify.com/drop](https://app.netlify.com/drop)**

### 단계 2: 폴더 올리기
1. 바탕화면에 있는 **`웹명함` 폴더** 전체를 마우스로 선택합니다.
2. 웹브라우저 창 화면 중앙의 **"Drag and drop your site folder here"** 라고 적힌 점선 박스 안으로 폴더를 끌어다 놓습니다.
3. 파일 업로드가 완료되기까지 약 3~5초간 기다립니다.

### 단계 3: 배포 완료 및 주소 확인
1. 화면에 초록색 글씨로 웹사이트 주소(예: `https://glowing-gelato-12345.netlify.app`)가 생성됩니다.
2. 해당 주소를 클릭하면 실제 작동하는 김이현님의 3D 웹 명함을 확인하실 수 있으며, 모바일 폰에서도 동일하게 접속이 가능합니다!

### 단계 4: (선택) 주소 이름 예쁘게 변경하기
1. 화면에 보이는 **[Sign up to manage this site]** 버튼을 눌러 회원가입 및 로그인을 진행합니다.
2. 로그인 후 대시보드 화면에서 **[Site configuration]** (사이트 설정) 버튼을 누릅니다.
3. **[Site details]** ➡ **[Change site name]** 버튼을 클릭합니다.
4. `yihyun-kim` 또는 `yihyun-card`와 같이 원하는 영문 사이트 주소를 적고 저장하면 주소가 깔끔하게 변경됩니다.

## ✍️ 나만의 정보로 내용 수정하기
텍스트나 프로필 사진을 바꾸고 싶다면 다음 가이드를 참고하세요:
1. **사진 바꾸기**: 원하는 본인 사진을 이 폴더에 넣고 이름을 `profile.png`로 동일하게 바꾼 뒤(기존 파일 덮어쓰기), 위의 Netlify Drop 사이트에 폴더를 다시 드래그 앤 드롭하면 즉시 갱신됩니다.
2. **글자 바꾸기**: 메모장이나 VS Code로 `index.html`을 열어서 **김이현**이나 **이메일 주소**, **경력 내용** 부분을 본인의 정보로 자유롭게 타이핑하여 수정한 후 저장하고 넷리파이에 폴더를 다시 드래그 앤 드롭하시면 즉시 업데이트됩니다.
